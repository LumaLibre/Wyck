When ran as an external plugin, BiomesAPI provides a small configuration for end users to
change the behavior of how BiomesAPI acts as a plugin.

The BiomesAPI configuration can be found at `~/plugins/BiomesAPI/settings.yml`.
Restart your server to apply any changes.
