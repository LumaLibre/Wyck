Worldgen rarely uses plain numbers. Instead it uses **value providers**, small objects that produce a number, a height,
or a vertical position, sometimes randomly. Carvers, placement modifiers, and feature configurations all take these
rather than raw values, so it is worth knowing the four you will reach for most.

## VerticalAnchor

A <SourceLink path="wrapper/worldgen/valueproviders/VerticalAnchor.java">VerticalAnchor</SourceLink> describes a single Y
position relative to the world. It is the building block for height ranges.

<CodeBlock
    lang="java"
    filename="ExamplePlugin.java"
    code={`import me.outspending.biomesapi.wrapper.worldgen.valueproviders.VerticalAnchor;

    VerticalAnchor.absolute(64);      // exactly Y=64
    VerticalAnchor.aboveBottom(8);    // 8 blocks above the world's minimum height
    VerticalAnchor.belowTop(1);       // 1 block below the world's maximum height`}
/>

## IntProvider

An <SourceLink path="wrapper/worldgen/valueproviders/IntProvider.java">IntProvider</SourceLink> produces an integer,
either constant or sampled from a distribution. Counts and offsets use these.

<CodeBlock
    lang="java"
    filename="ExamplePlugin.java"
    code={`import me.outspending.biomesapi.wrapper.worldgen.valueproviders.IntProvider;

    IntProvider.constant(4); // always 4
    IntProvider.uniform(2, 8); // evenly between 2 and 8
    IntProvider.triangle(6); // triangular distribution centered with spread 6`}
/>

## FloatProvider

A <SourceLink path="wrapper/worldgen/valueproviders/FloatProvider.java">FloatProvider</SourceLink> is the floating-point
equivalent, used for radii, scales, and probabilities inside carvers and features.

<CodeBlock
    lang="java"
    filename="ExamplePlugin.java"
    code={`import me.outspending.biomesapi.wrapper.worldgen.valueproviders.FloatProvider;

    FloatProvider.constant(3.0f); // always 3.0
    FloatProvider.uniform(0.1f, 0.9f);  // evenly between 0.1 and 0.9
    FloatProvider.trapezoid(0.0f, 6.0f, 2.0f); // trapezoidal distribution`}
/>

## HeightProvider

A <SourceLink path="wrapper/worldgen/valueproviders/HeightProvider.java">HeightProvider</SourceLink> produces a Y
position by combining vertical anchors. Height ranges in carvers and placement use these.

<CodeBlock
    lang="java"
    filename="ExamplePlugin.java"
    code={`import me.outspending.biomesapi.wrapper.worldgen.valueproviders.HeightProvider;
import me.outspending.biomesapi.wrapper.worldgen.valueproviders.VerticalAnchor;

HeightProvider.uniform(VerticalAnchor.aboveBottom(8), VerticalAnchor.absolute(180));
HeightProvider.trapezoid(VerticalAnchor.absolute(0), VerticalAnchor.absolute(128), 0);`}
/>
