#### <SourceLink path="keys/ResourceKey.java">ResourceKey</SourceLink>

Represents a key for a biome resource in the game.
This interface extends Kyori's [`Keyed`](https://jd.advntr.dev/key/4.26.1/net/kyori/adventure/key/Keyed.html) interface, allowing it to be used as a key in various registries and maps.

* <SourceLink path="keys/ResourceKey.java">of(String, String)</SourceLink>: Creates a new ResourceKey with the specified namespace and value.
* <SourceLink path="keys/ResourceKey.java">fromString(String)</SourceLink>: Creates a new ResourceKey from a string in the format "namespace:value".
* <SourceLink path="keys/ResourceKey.java">namespace()</SourceLink>: Returns the namespace of this ResourceKey.
* <SourceLink path="keys/ResourceKey.java">path()</SourceLink>: Returns the path of this ResourceKey.
