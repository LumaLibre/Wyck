This is a wrapper for Minecraft's internal `SimpleParticleType`. Minecraft biomes only support simple particles as their ambient particles.
That means redstone dust, entity effects, spell particles, or any other particle that requires data to be passed in will not work.

In order to avoid NMS usage, BiomesAPI wraps around all of Minecraft's internal `SimpleParticleType`s.


`AmbientParticle`s are intended to be used alongside the [`ParticleRenderer`](/guides/components/particle-renderer)
