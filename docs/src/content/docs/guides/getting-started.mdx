# What is BiomesAPI?

BiomesAPI is a powerful library for creating custom biomes on `1.21.11+` [Paper](https://papermc.io/) based servers in Minecraft.
It provides an easy-to-use API for defining and registering biomes, allowing developers to change the look of a Minecraft block, chunk, or world with just a few steps!
BiomesAPI even supports sending custom biomes to clients using ProtocolLib! See our [**packets**](/guides/sending-biomes-with-packets/) tutorial for more information.

The latest version at the time of writing is `0.0.9`, see: https://repo.jsinco.dev/#/releases/me/outspending/biomesapi/BiomesAPI to find the latest!

# Shading

Please note that you will need to **shade** BiomesAPI into your plugin for the library to actually work. If you're using Gradle I'd recommend using GradleUp's ShadowJar and the maven-shade-plugin if you choose to use Maven.
