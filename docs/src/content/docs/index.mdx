BiomesAPI is a custom biome library for `1.21.11+` Paper servers. It is a fork of the original
[BiomesAPI](https://github.com/Outspending/BiomesAPI) project by Outspending.


## 1.21.11+ Only!

BiomesAPI only supports `1.21.11+` **Paper-based** servers. Support is not guarenteed anywhere else!


## Sending Biomes with Packets

BiomesAPI can send biomes using [ProtocolLib](https://github.com/dmulloy2/ProtocolLib). 
See our [**packets**](/guides/sending-biomes-with-packets/) tutorial for more information.


## Documentation

This documentation site contains guides and API references to help you get started with BiomesAPI.
See our [**getting started**](/guides/getting-started/) guide to set up BiomesAPI in your project!
