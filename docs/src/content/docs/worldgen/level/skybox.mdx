A <SourceLink path="wrapper/level/dimension/Skybox.java">Skybox</SourceLink> is
a visual effect that can be applied to a dimension to change the appearance of the sky in a given world.

There are three skybox options available in BiomesAPI:

- **Overworld**: The default skybox used in the Overworld dimension, featuring a blue sky with clouds and a sun.
- **End**: The skybox used in the End dimension.
- **None**: No skybox is applied, resulting in a completely blank sky.
