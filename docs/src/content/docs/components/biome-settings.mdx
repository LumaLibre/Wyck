#### <SourceLink path="wrapper/BiomeSettings.java">BiomeSettings</SourceLink>

A record holding the environmental settings for a custom biome, such as depth, scale, temperature, downfall, and precipitation. Instances are created either with sensible defaults or through the nested fluent `Builder`.

Each numeric property is bounded to a <SourceLink path="wrapper/BiomeSettings.java" line={18}>`@Range(from = 0, to = 25)`</SourceLink>.

* <SourceLink path="wrapper/BiomeSettings.java" line={34}>builder()</SourceLink>: Creates a new `Builder` for constructing a `BiomeSettings` instance.
* <SourceLink path="wrapper/BiomeSettings.java" line={46}>defaultSettings()</SourceLink>: Returns a `BiomeSettings` with default values: depth `0.1`, scale `0.2`, temperature `0.5`, downfall `0.5`, modifier `NONE`, and precipitation enabled.

#### <SourceLink path="wrapper/BiomeSettings.java" line={59}>BiomeSettings.Builder</SourceLink>

A fluent builder for `BiomeSettings`. Each setter returns the builder for chaining, and <SourceLink path="wrapper/BiomeSettings.java" line={153}>build()</SourceLink> produces the final record. Any property left unset falls back to its default.

* <SourceLink path="wrapper/BiomeSettings.java" line={76}>depth(float)</SourceLink>: Sets the biome depth. Default `0.1`.
* <SourceLink path="wrapper/BiomeSettings.java" line={89}>scale(float)</SourceLink>: Sets the biome scale. Default `0.2`.
* <SourceLink path="wrapper/BiomeSettings.java" line={102}>temperature(float)</SourceLink>: Sets the biome temperature. Default `0.5`.
* <SourceLink path="wrapper/BiomeSettings.java" line={115}>downfall(float)</SourceLink>: Sets the biome downfall. Default `0.5`.
* <SourceLink path="wrapper/BiomeSettings.java" line={128}>modifier(BiomeTempModifier)</SourceLink>: Sets the temperature modifier. Default `BiomeTempModifier.NONE`.
* <SourceLink path="wrapper/BiomeSettings.java" line={141}>hasPrecipitation(boolean)</SourceLink>: Sets whether the biome has precipitation. Default `true`.
* <SourceLink path="wrapper/BiomeSettings.java" line={153}>build()</SourceLink>: Builds and returns a new `BiomeSettings` from the configured values.

### Example Usage
<CodeBlock
    lang="java"
    filename="ExamplePlugin.java"
    code={`import wrapper.dev.wyck.BiomeSettings;
import environment.wrapper.dev.wyck.BiomeTempModifier;
import org.bukkit.plugin.java.JavaPlugin;

public class ExamplePlugin extends JavaPlugin {
    @Override
    public void onEnable() {
        // The quickest option, using vanilla defaults.
        BiomeSettings defaults = BiomeSettings.defaultSettings();

        // Or customize individual properties via the builder.
        // Omit an option to use the default value for that property.
        BiomeSettings settings = BiomeSettings.builder()
            .depth(0.15F)
            .scale(0.25F)
            .temperature(0.8F)
            .downfall(0.4F)
            .modifier(BiomeTempModifier.FROZEN)
            .hasPrecipitation(true)
            .build();
    }
}`}
