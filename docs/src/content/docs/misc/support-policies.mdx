Wyck compiles heavily against Paper and its internals. As such, we only provide support for Paper and a handful of its forks.
It is possible that Wyck may work on other server software, but we do not explicitly provide support for any software other than those listed below.

| Software | Supported Versions | Supported |
| -------- | ------------------ | --------- |
| [Paper](https://papermc.io/software/paper) | **1.21.11, 26.1.2, 26.2** | ✅ |
| [Purpur](https://purpurmc.org) | **1.21.11, 26.1.2, 26.2** | ✅ |
| [Folia](https://papermc.io/software/folia) | **1.21.11, 26.1.2, 26.2** | ✅ |
| [Canvas](https://canvasmc.io) | **1.21.11, 26.1.2, 26.2** | ✅ |
