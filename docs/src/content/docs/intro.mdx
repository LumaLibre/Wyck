BiomesAPI is an API for plugins to create custom biomes, dimensions, and world generation features
without utilizing datapacks. BiomesAPI is designed for Paper and Folia-based servers and allows you to easily
modify the game world without touching any internals.

## Custom Biomes

BiomesAPI's predominant feature is the ability to create custom biomes.
With our wrappers, you define every single property of a biome exactly as it appears in the game and
render them to clients through vanilla means, or by using our packet-based renderers provided in the API.

To get started with creating and rendering custom biomes, follow our guides:

- [**Creating Biomes**](/biomes/custom-biome/) - Learn how to create custom biomes with BiomesAPI.
- [**Rendering Biomes**](/biomes/biome-setter/) - Learn how to render custom biomes to players using BiomesAPI.
- [**Wrappers**](/wrappers/environment-attributes/) - Learn about the wrappers provided for creating custom objects.
- [**Biome World Generation**](/worldgen/biome-generation-settings/) - Learn how to customize (biome) world generation with BiomesAPI.

## Custom Dimensions

In addition to custom biomes, BiomesAPI also allows you to create custom dimensions.
Our docs have extensive guides on how to use our custom wrappers to create just about anything.

Getting started with dimensions and levels:

- [**Creating Dimensions**](/worldgen/level/dimensions/) - Learn how to create custom dimensions with BiomesAPI.
- [**LevelCreators**](/worldgen/level/dimensions#levelcreators) - Learn how to create custom worlds with BiomesAPI.
